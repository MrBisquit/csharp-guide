# A complete beginners guide to C#

A complete beginners guide to C#, and the fundamentals of programming with it.

## About & How to use

I suggest forking or cloning this repository, and ticking each of the items off.
There are checkboxes next to each of the sections.

My favourite way of learning, especially with programming, is to FAFO
(~~Fu~~ Mess Around and Find Out). Of course, everyone has a different way of
learning, but I still encourage you to create a project, and learn the language
along the way. People think that you can learn programming through just doing
small challenges. While yes, this may help you, this isn't the most efficient
way of learning programming, you need the experience with an actual project.
So you get to learn how to program, and then come out with a working program
at the end.

**No LLM (GenAI) was used in the creation of this guide**
